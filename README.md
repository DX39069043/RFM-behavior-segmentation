# 电商用户画像：基于RFM模型与行为的精细化用户分层

## 一、项目背景与目的

### 1. 项目背景 

在当前的电商存量竞争时代，依靠粗放式买量获取增长的红利已经见顶，如何对庞大的现有客群进行精细化运营与价值挖掘，成为了提升平台营收的核心突破口。

然而，在实际的业务落地中，传统的数据分层策略面临着以下两大痛点：

- **二八法则陷阱：** 真实的电商大盘数据往往呈现长尾偏态分布，超过 80% 的用户属于“零购买”或“低频低额”的底层基数人群。传统的 RFM 模型（仅依赖 R最近消费、F消费频次、M消费金额）会粗暴地将这 80% 的人全部归为“低价值用户”，导致业务团队无法从庞大基数中发掘高潜力人群。
- **后置指标的局限性：** 传统的 RFM 指标是典型的“后置结果指标”，只能反映用户“已经买了什么”，却无法刻画用户“买之前的意愿与阻力”。例如，同样是购买过 1 次的用户，一个是“目标明确、搜完即买”，另一个是“反复比较了 100 个页面才勉强下单”，传统RFM模型无法区分两者的决策差异，也就无法针对这两种用户

### 2. 项目目的

为了打破传统 RFM 模型的业务局限性，本项目旨在借助数据埋点技术，通过引入前端交互行为数据，重构一套“传统RFM模型分层 + 行为细分”的双重分层模型，将“事后描述”转化为“事前干预”。具体目的如下：

- **引入行为指标：** 打破仅依靠消费金额打标的单一维度，深度挖掘用户的前端交互行为（浏览页数、有效停留时长）。构建“探索度”与“摩擦力”两大衍生指标，还原用户的真实购物心理与行为画像。
- **精准圈人与战略矩阵裂变：** 采用一维 K-Means 聚类自适应寻找业务动态阈值，在传统 8 大 RFM 分群的基础上进行二次裂变，构建 5 大战略人群矩阵。重点实现：
  - **高M值群体的排雷：** 精准识别出高 M 值人群中处于转化受阻状态的“高危异动VIP”。
  - **低M值群体的掘金：** 从庞大的底层灰色客群中，提纯出具备高探索度、高摩擦力的“高潜力客户”。



## 二、数据来源

本项目的数据集来自 Kaggle 平台上的两个真实大型电商行为数据集。为了确保分析模型不仅在理论上逻辑自洽，且在实际商业环境中具备落地价值，本项目采用了“双轨验证”的数据处理与应用策略：

### **数据集一：特征逻辑与指标有效性验证**

- **数据集名称:** E-commerce Product Recommendation Collaborative
- **数据来源:** Kaggle (https://www.kaggle.com/datasets/kartikeybartwal/ecommerce-product-recommendation-collaborative)
- **项目应用:** 该数据集主要应用于项目初期的探索性数据分析 (EDA) 与特征工程构建阶段。由于其包含了丰富的用户交互与商品反馈信息，本项目利用该数据成功推导、测试了本项目核心创新的两大前置行为指标——“探索度”与“摩擦力”的统计学合理性。它从数据底层证实了本项目的核心假设：“高探索度并不必然指向高转化，而高摩擦力往往预示着核心用户的决策受阻”。

### **数据集二：模型商业价值验证**

- **数据集名称:** e-Commerce Behavior Data from Multi Category Store
- **数据来源:** Kaggle (https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store)
- **项目应用:** 该数据集包含千万级的大量电商用户完整时间序列行为日志（包括 view, purchase 等事件及精准时间戳）。该数据的高度真实性能够很好的测试本项目的商业价值，具体操作有：
  1. **时间序列切分:** 提取 10 月份数据作为特征观察期进行用户画像打标，将 11 月份数据作为效果验证期进行真实业绩回溯。
  2. **用户分层:** 在该数据集上实际运行自适应 K-Means 阈值算法，对百万级用户进行精细划分。
  3. **增量测算:** 基于该数据集计算出不同分层人群的基准客单价 (ARPPU)，从而严谨地推演出精细化营销干预下（如针对“高潜力客户”）能够为大盘带来的纯增量业绩与增长百分点。

## 三、数据分析与可视化

### 1. 构建前置行为指标

传统电商日志往往只记录事件的时间戳（Event Time），缺乏精确的停留时长。为了准确量化用户的购物意愿与阻力，本项目首先进行了序列特征提取：

- **会话时长估算：** 对单用户的行为序列按时间差进行切分，将间隔小于 30 分钟的连续动作判定为有效停留，累加得出估算时长，弥补了底层数据的维度缺失。
- **探索度：** 结合“浏览页数”与“估算时长”，科学衡量用户在购买前的活跃度与探索广度。
- **摩擦力：** 公式为 `浏览页数 / (购买次数 + 1)`。该指标刻画了用户“每完成一次交易所需要付出的浏览成本”，摩擦力越高，代表决策链路越长、转化受阻越严重。

### 2. 阈值切分

在确定人群边界时，放弃了人为决定的硬性阈值，采用机器学习：

- **分布形态验证：** 对标准化前后的 RFM 及探索度、摩擦力得分进行精细化直方图绘制（自适应 X 轴边界）。观察发现，电商数据呈现极端的右偏长尾分布。
- **一维 K-Means 聚类：** 针对长尾分布，采用一维 K-Means 算法对连续型得分进行无监督聚类，自动寻找数据分布中的“自然断层”作为阈值（如探索度界线、摩擦力界线）。这保证了分类标准随大盘数据的变化而动态调整，具备极强的模型泛化能力。

### 3. 精细化用户分层

在传统的 8 大 RFM 基础人群之上，串联探索度与摩擦力，构建出极具业务指导意义的精细用户画像：

- **针对高客单价 的群体：**
  - 高摩擦力 = **高危异动VIP**（陷入选择困难或遇到支付阻碍，需立即客服介入）。
  - 低摩擦力 + 高探索度 = **深度互动用户**（高粘性、爱逛爱买，适合推新品）。
  - 低摩擦力 + 低探索度 = **精准直购用户**（目的性极强，搜完即买，适合推周期复购品）。
- **针对低价值/下沉群体：**
  - 高摩擦力 + 高探索度 = **高潜力客户**（极度活跃但尚未大额转化，是破冰营销的重要目标）。

### 4. 可视化

在可视化洞察阶段，本项目解决了一个经典的电商分析痛点：“二八法则导致的视觉坍塌”。

- **痛点：** 真实大盘中超过 80% 是“常规/其他类用户”。如果使用传统的绝对人数条形图，极具商业价值的 VIP 细分人群（仅占少数）会被巨大的灰色柱状体彻底挤压，导致图表失去视觉焦点。
- **解决方案 ：** 本项目采用 `Pandas` 交叉表结合 `Matplotlib`，绘制了 100% 横向堆叠条形图。将横轴从“绝对人数”强制转换为“组内浓度占比 (0-100%)”。
- **结果：**经可视化呈现，彻底排除了庞大基数人群的干扰，管理者可以一眼看清各个 RFM 阶层内部的“高危异动”与“高潜”人群的真实纯净浓度，为后续的资源倾斜提供了直观的数据级信服力。



## 四、最终总结与业务建议

基于本模型定位出的四大目标人群，针对其不同的行为特征，提出以下精细化运营建议与策略缘由：

#### 1. 高潜力客户

- **人群特征：** 该群体在传统模型中通常被归为“低价值”用户。但本模型敏锐捕捉到他们具有**高探索度**与**高摩擦力**。这说明他们对平台或者对某一商品有极大的兴趣，浏览活跃但有明显的决策障碍（例如觉得心仪的商品价格略贵，或者担心平台售后服务不够周到），缺乏一个“跨过购买门槛”的推力。
- **业务建议：**
  - **精准破冰营销：** 放弃大水漫灌的通用折扣，针对该群体定向发放“限时大额首单券”或“品类专属破冰券”，制造购买紧迫感。
  - **个性化低门槛引流：** 根据其高频浏览的商品轨迹，精准推送对应的“低客单价引流款（福利品）”，先完成首单转化，培养支付习惯。
  - **商业预期：** 本项目在商业价值验证阶段已证实，该群体的自发转化率（无营销干预）为37%，远高于一般低M值用户的平均转化率（11%）。仅需 50% 的促转成功率，即可撬动极其可观的大盘纯增量业绩。

#### 2. 高危异动VIP

- **人群特征：** 核心高净值客户，但当前表现出极高的**摩擦力**。这意味着曾经高消费的 VIP客户，现在却出现了“反复浏览却迟迟不下单”、“频繁加购后放弃支付”的异常受阻状态。这极大概率是遇到了糟糕的购物体验、缺货、或者是其他竞品提供了更好的服务。
- **业务建议 ：**
  - **障碍排查：**提取该群体的行为日志进行专项漏斗分析；或查询近期是否有投诉记录
  - **主动干预：** 安排专属客服进行 1V1 定向私信，主动询问受阻原因并提供解决方案。



#### 3. 深度互动用户

- **人群特征：** 高客单价、高探索度、低摩擦力。这是平台的忠实用户。他们不仅极具购买力，而且热爱在平台上“逛”，愿意投入大量时间浏览不同页面，对平台有极高的忠诚度和探索欲，是试错成本最低、最容易接受平台新品和高溢价商品的群体
- **业务建议 ：**
  - **跨品类交叉销售 ：** 不要局限于他们以往购买的商品，应大幅拓宽推荐品类。例如在支付页面添加“顺便买”的推荐商品。
  - **深度参与社区生态：** 邀请他们加入核心内测群，或鼓励他们产出商品评价、买家秀。利用他们的高参与度来反哺平台的社区内容生态，鼓励用户拉新
  - **付费会员转化：** 极力向该群体推销 PLUS/VIP 付费会员制，用“沉没成本”锁定其长期生命周期价值。

#### 4. 精准直购用户

- **人群特征：** 高客单价，但**低探索度**与低摩擦力。这类用户目的性极强，通常是“搜索-加入购物车-付款-离开”一气呵成。他们极其看重时间效率，反感繁琐的营销套路和无意义的页面跳转。
- **业务建议：**
  - **极致的极简体验：** 在其核心链路上减少弹窗干扰，提供“一键复购”或“快捷支付”功能，确保其直购通道绝对顺畅。
  - **周期性静默推荐：** 针对其购买的消耗型商品（如个护、母婴、宠物用品），计算其使用周期，在即将耗尽前进行“低打扰”的精准补货提醒（如**首页推荐**，而不是短信广告或者需要手动关闭的弹窗）。
  - **避免过度打扰：** 严禁向该群体高频推送与其历史购买无关的促销信息，以免引发反感导致卸载。
